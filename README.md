# 🧬 BioMed Explorer — Educational Game

> An interactive web game that teaches biomedical engineering concepts to young learners through four mini-games.

[![Play](https://img.shields.io/badge/Play-Live-green.svg)](https://mymunah-07lmtc.github.io/biomed-explorer/)  
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

---

## 🎮 Mini-Games

| Game | Description |
|------|-------------|
| **ECG Simulator** | Adjust BPM and watch the heartbeat waveform animate |
| **Immune System Defender** | Wave-based arcade game: click germs, collect shields, build combos |
| **Prosthetic Designer** | Design a prosthetic with weight, durability, and cost trade-offs |
| **Body Systems Quiz** | Answer biomedical trivia with immediate feedback |

---

## 🛠️ Tech Stack

- **HTML/CSS/JS** (vanilla, no frameworks)
- **Canvas API** for animations
- **Web Audio API** for sound effects
- **Single HTML file** — no dependencies, works offline
